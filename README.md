#Определение
Git это - программа, позволяющая отследить историю изменений  


#основные команды

 
---  
##Посмотреть моё местоположение  
Эту функцию выполгяет команда pwd
---


---  
##Отобразить папки текущей директории
Это можно сделать командой ls  


###Дополнительные возможности ls  


####Отобразить все скрытые файлы  
Для этого вводится ls -a, что также называют ls с флагом  


####Показать папки родительской директории  
ls ..  


####Показать папки домашней директории  
ls~  
---


---  
##Вернуться в родительскую директорию    
cd ..  
---


---  
##Создать файл  
touch + название файла с тире вместо пробелов  
touch ../../file.txt создать файл на 2 дирректории выше
---


---  
##Создать репозиторий  
mkdir + название  
  
mkdir -p dir1/dir-iside/dir-deeper-inside создаст папку dir-deeper-inside в папке dir-iside,  
которая находится в папке dir  
---


---  
##Скопировать файл  
вводим команду cp, затем файл, который жедаем скопировать, и через пробел вводим место, куда хотим скопировать  
---


---  
##Перемещение файлов и папок  
mv + папка, или файл, который хотим переместить  
---


---  
##Прочитать файл  
Cat + название файла(только с файлом text)  
---


---  
##Удаление  
###Удалить файл  
rm


###Удалить папку(если она пустая)  
rmdir


###Удалить папку со всеми файлами  
rm -r  
---


---  
##Сделать папку git репозиторием  
git init  
---


---  
##Разгидить папку  
Переходим в папку с помощью команды cd  
затем вводим комманду rm -rv.git  
---


---  
##Проверить статус  
git status  
---


---
##Подготовка к сохранению в репозиторий  
###Подготовить все файлы в репозитории  
git add --all


###Добавить отдельный файл  
git add


###Добавить всю текущюю папку  
git add.  
---


---  
##Создать коммит  
git commit -m + 'Название коммита'  
---


---  
##Показать историю коммитов  
git log  
---


---  
##Передать коммит в удаленный репозиторий при уже пройденой синхронизации  
git push  
---  


---  
##Скопировать проект  
git clone  
---
